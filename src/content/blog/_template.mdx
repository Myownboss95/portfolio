Open with the point. One short paragraph that says what this post argues, so a
reader knows in ten seconds whether to keep going.

## A section heading

Plain Markdown below the frontmatter. Use `##` for sections and `###` for detail
inside a section. Fenced code blocks work:

```ts
const answer = await retrieve(query);
```
